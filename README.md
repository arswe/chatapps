# Chat application using Node.js, Express, and Socket.io.

## Installation

add dotenv file with the following variables:
PORT=5000
MONGO_URI=mongodb://localhost:27017/chat
JWT_SECRET=secret
JWT_EXPIRES_IN=90d
JWT_COOKIE_EXPIRES_IN=90
NODE_ENV=development

## Usage

npm start
